■1. 作品概要
現実の麻雀牌を画像認識で読み込む「AI推論モード」と、AIの思考を学びながら遊べる「対戦モード」の2つのシーンを備えた、ブラウザ上で動作する麻雀アプリケーションです。
単なる既存ライブラリの組み合わせではなく、「画像認識モデルの独自育成」と「麻雀の複雑な状態を表現するテンソル構造の設計」をゼロから行った点が最大の特徴です。

・開発環境：Unity (C#) ※WebGLビルド、Python (AIモデル学習用)
・主要技術：YOLOv8（画像認識）、深層学習によるAI推論

■2. 2つの主要機能（シーン）
① AI推論モード
　Webカメラ等で現実の手牌を読み込み、AIが最適な有効打牌を推論して提示します。手牌の画像認識には、自ら745枚の画像にアノテーションを行って訓練したYOLOv8モデルを使用し、96.6%の識別精度を実現しています。

② 対戦モード＆振り返り分析機能
　CPUと対戦できるモードです。単に遊ぶだけでなく、終局後の「振り返り」にて、対局中の全選択から「プレイヤーの選択とAIの推奨打牌が最も乖離した場面 Top3」を抽出し、AIの推奨打牌とともに提示します。これにより、プレイヤーのスキルアップを促す体験設計としています。

■3. 技術的なアピールポイント
オンライン麻雀「天鳳」の上位卓ログから約12万件の学習サンプルを抽出。麻雀特有の複雑な盤面情報をAIに正しく学習させるため、独自のテンソル構造（22チャンネル×34次元）を自ら設計しました。137通りの選択肢の中から、上位プレイヤーと約64%の確率で打牌が一致する精度を達成しています。

■5. 実行・操作方法
環境構築やインストールは不要です。

1. index.htmlを、Google Chrome や Microsoft Edge などのWebブラウザで開いてください。
2. タイトル画面から、遊びたいモード（シーン）をクリックして進行してください。

※「AI推論モード」でカメラをご用意できない環境の場合は、画面上の[〇〇ボタン]から、同梱の /sample_images 内の画像を読み込ませることで推論をテストできます。

■6. 最後に
複雑なルールを持つ麻雀というゲームにおいて、「いかにして現実の情報をデータ化し、AIに正しく評価させるか」というロジックの構築、そしてそれを「プレイヤーが学びを得られる形（UI/UX）」に落とし込むことに最もこだわりました。
